<h1>How to use</h1>
<h3>clone this repository in your computer or create a new repository using this template</h3>
<h3>then open in vs code</h3>

<h3>go to server path and run "npm i" or npm i express cors</h3>

<h3>go to client path and run "npm i"</h3>


<h3>now you can run server and client in your browser. to run give command in cmd -></h3>

<h3>go to server path and run "nodemon index.js"</h3>
<h3>go to client path and run "npm run dev"</h3>
